# Анализ параметров системы

## Описание

В интерфейсе присутствует блок настроек системы, включающий платформы:

- ВКонтакте
- Telegram
- 2GIS
- Яндекс

---

## Доступные режимы

Для каждой платформы:

- Работать автономно
- Согласовывать позитив
- Согласовывать нейтрал
- Согласовывать негатив

---

## Бизнес-логика

- Активен только один режим
- Для изменения необходимо снять текущий
- Состояния должны быть взаимно исключающими

---

## Тест-кейсы

- Проверка переключения режимов
- Проверка невозможности выбора нескольких режимов
- Проверка сохранения настроек
- Проверка отображения текущего состояния

---

## ⚠️ Риски

- Возможность включить несколько режимов одновременно
- Рассинхрон UI и backend
- Потеря настроек при обновлении страницы
